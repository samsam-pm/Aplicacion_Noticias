Aplicación de Noticias con Ionic 6

Desarrollada con Ionic 6 y Angular, esta aplicación híbrida y PWA permite a los usuarios visualizar noticias en tiempo real desde una API externa. Implementa un diseño modular, optimización para diferentes dispositivos y una interfaz completamente responsiva.

Características principales:

- Integración con servicios para consumir APIs de noticias.

- Implementación de Lazy Loading y paginación con Infinite Scroll.

- Diseño responsivo con Ionic Grid y CSS Flexbox.

- Uso de modales e iconos interactivos para mejorar la UX.

Tecnologías:

- Ionic 6 y Angular

- Firebase Hosting para despliegue.

- PWA para funcionalidad offline y rendimiento mejorado.

Objetivos:

- Crear una interfaz intuitiva utilizando ion-card, ion-button, ion-input, etc.

- Implementar servicios para gestionar datos y manejar la lógica de negocio.

- Validar la responsividad en móviles, tabletas y escritorios.
